# Inbuilt data structures and input/output
## CoreModules
### imgcodecs
    Handles reading and writing of image files.
### videoio
    Handles everything related to the input and output of video files.
### imgproc
    Do things such as image filtering, morphological operations, geometric transformations, color conversions, drawing on images, histograms, shape analysis, motion analysis, feature detection and more.
### ximgproc
    With contains advanced image processing algorithms, such as, structured forests for edge detection, domain transform filter, adaptive manifold filter and so on.
### highgui
    Handles all the high-level user interface operations.
### video
    Analyzing the motion between successive frames in a video.
    Tracking different objects in a video.
    Creating models for video surveillance and so on.
### videostab
    Deals with video stabilization.
### calib3d
    3D reconstruction. Find the relationship between various objects in those 2D images to compute their 3D positions.
### feature extraction: features2d and xfeatures2d and bioinspired
    Provides functions to deetct and extract all these features.
    *xfeatures2d* *bioinspired*
### Object detection: objdetect and xobjdetect
    Refers to detecting the location of an object in a giving image.
### Machine Learning: KNN, SVM, FLANN
### Computational photography
    photo and xphoto module, *stitching* to create panoramic images
### Shape Analysis
    *shape* module provides all the algorithms reuired to extract different shapes, measure similarity between them, transform the shapes of objects and more.
### Optical flow algorithms
    Are used in videos to track features across successive frames. *optflow* module contains all the algorithms. Also a module called *tracking* that contains more algorithms that can be used to track features.
### Face and object recognition
    *face* module deals with face recognition
    *saliency* is designed to find salient regions and objects in the images that can help with different applications.
### Surface matching
    *surface_matching* that contains algorithms for 3D object recognition and a pose estimation algorithm using 3D features.
### Text Detection and recognition
    *text* module
